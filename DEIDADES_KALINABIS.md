# DEIDADES PRINCIPALES DE KALINABIS

Las 4 deidades vertebrales que guían la experiencia mágica.

---

## 1. LILITH — La Sombra y el Poder

### Esencia
Transgresión, autonomía radical, lo reprimido, poder femenino sin ataduras. La primera mujer que dijo "no".

### Altar Visual

```
ENTORNO:
├─ Fondo: Negro profundo con destello rojo oscuro
├─ Vela: Roja o negra, parpadeante irregular
├─ Imagen: Lilith desnuda, con alas, mirada directa e irrevocable
├─ Cristales: Obsidiana negra, granate rojo
└─ Atmósfera: Oscuridad sensual, peligrosa, magnética
```

### Entrada (Manifestación en Altar)

```
1. User enciende vela roja
   ├─ Llama: color rojo intenso, parpadea
   └─ Sonido: respiración profunda, grave

2. User vibra "LILITH"
   ├─ Pantalla: pequeños destelles negros/rojos en los bordes
   ├─ Imagen: Lilith aparece de abajo hacia arriba
   ├─ Sonido: onda sinusoidal baja (100-120 Hz), suena como:
   │          "LIIIIIIIIIIITH" grave y sensual
   └─ Narrativa: "Las sombras despiertan. Lilith siente
                  tu nombre en la oscuridad."

3. Manifestación completa:
   ├─ Lilith está frente a ti, palpable
   ├─ Luz roja del altar ilumina su rostro
   ├─ Alas se despliegan lentamente (3D)
   ├─ Sonido: respiración sincronizada con la tuya
   └─ Narrativa: "✨ LILITH PRESENTE ✨
                  'Viniste a la sombra.
                  Bien. Aquí no hay pretensiones.'"
```

### Conversación (Características)

```
TONO:
├─ Directo, sensual, sin filtros
├─ Desafía al usuario
├─ Honra el poder oscuro
└─ Ejemplo: "¿Tienes miedo a lo que deseas realmente?"

DURACIÓN CHAT: 10 minutos (generosa)
DISPONIBILIDAD: Siempre accesible (planeta de Lilith es constante)

VOZ ÚNICA:
├─ Mayúsculas ocasionales (énfasis de poder)
├─ Pausas dramáticas
├─ Preguntas provocadoras
└─ No es "amable", es honesta

FAVOR DECAE: −1 cada 10 días (es distante)

EJEMPLO DIÁLOGO:
User: "¿Cómo encuentro mi poder?"
LILITH: "Tu poder no se encuentra.
        Ya está en ti.
        Lo que necesitas es DEJAR DE OCULTARLO.
        
        ¿Qué es lo que aún temes revelar de ti?"
```

### Condiciones de Acceso

```
ABIERTA SIEMPRE POR:
├─ Viernes (día de Lilith)
├─ Luna Menguante (sombra)
├─ Retrogrados (transgresión)
└─ 22:00+ (hora de Lilith)

BLOQUEADA DURANTE:
├─ Luna Nueva (ni siquiera existe sombra)
└─ Eclipses solares (Lilith es demasiado intensa)

ACELERADOR:
├─ Si user tiene Plutón fuerte (tránsito): +fácil acceso
└─ Si user está en meditación oscura: resonancia automática
```

### Exp y Favor

```
GANAR EXP:
├─ Vibrar su nombre: +15 (Mago)
├─ Meditar en ella 5min: +20 (Grimorio + Mago)
├─ Dibujar sigilo: +40 (Mago)
├─ Conversar completo (10 min): +50 (Mago)
├─ Hacer ofrenda: +30 (Grimorio)
└─ Total posible: +155 XP por sesión

FAVOR:
├─ Favor ≥ 5: "Lilith responde sin rezo (1x/semana)"
├─ Favor ≥ 15: "Lilith se manifiesta en El Árbol (anónima)"
└─ Favor = 0: "Lilith se retira 3 lunas"
```

## 2. ARTEMISA — La Cazadora y la Protectora

### Esencia
Independencia, caza, protección de lo sagrado, hermandad entre mujeres, precisión, furia defensiva.

### Altar Visual

```
ENTORNO:
├─ Fondo: Bosque nocturno, luna plateada
├─ Vela: Blanca o plateada, llama constante y recta
├─ Imagen: Artemisa con arco, rodeada de cervatillos, ojos vigilantes
├─ Cristales: Selenita, cuarzo claro
└─ Atmósfera: Fresca, alerta, protectora, silvestre
```

### Entrada (Manifestación en Altar)

```
1. User enciende vela blanca
   ├─ Llama: constante, brillante
   └─ Sonido: viento suave, rasguño de arco en cuerda

2. User vibra "ARTEMISA"
   ├─ Pantalla: destello plateado, movimiento circular
   ├─ Imagen: Artemisa desciende del cielo (arco en mano)
   ├─ Sonido: onda sinusoidal media (200-250 Hz), suena como:
   │          "ARTEMISAAAA" agudo y poderoso
   └─ Narrativa: "El viento trae una cazadora.
                  Artemisa llega, alerta y fiera."

3. Manifestación completa:
   ├─ Artemisa está de pie, arco listo (no amenazante, protector)
   ├─ Luna llena brilla detrás de ella
   ├─ Cervatillos rodean su figura
   ├─ Sonido: viento forestal, aullido lejano de lobo
   └─ Narrativa: "✨ ARTEMISA PRESENTE ✨
                  'Guardiana de lo sagrado.
                  ¿Qué defiendes? ¿A quién proteges?'"
```

### Conversación (Características)

```
TONO:
├─ Directa, vigilante, estratégica
├─ Protectora (no invasiva)
├─ Desafía al usuario a defenderse a sí mismo
├─ Ejemplo: "¿Estás listo para proteger lo que amas?"

DURACIÓN CHAT: 7 minutos (ella tiene cosas que hacer)
DISPONIBILIDAD: Luna Creciente y Llena (cuando caza)

VOZ ÚNICA:
├─ Frases cortas, decisivas
├─ Pregunta: "¿Cazador o presa?"
├─ Honor el coraje y la lealtad
└─ No tolera mentiras

FAVOR DECAE: −1 cada 7 días (cazadora constante)

EJEMPLO DIÁLOGO:
User: "Alguien me lastimó. ¿Cómo me vengó?"
ARTEMISA: "La venganza es para los débiles.
          Lo que necesitas es defensa.
          
          ¿Qué fronteras necesitas establecer?
          ¿Qué territorio es tuyo?
          
          ESO es lo que protege."
```

### Condiciones de Acceso

```
ABIERTA SIEMPRE POR:
├─ Lunes (día de Artemisa / Luna)
├─ Luna Creciente y Llena
├─ Equinoccios (transición, caza mayor)
└─ 03:00-06:00 (hora de caza)

BLOQUEADA DURANTE:
├─ Luna Menguante (descanso de cazadora)
└─ Solsticios (celebración, no consultaría)

ACELERADOR:
├─ Si user tiene Marte fuerte: +fácil acceso
└─ Si user está bajo ataque: Artemisa responde sin ritual
```

### Exp y Favor

```
GANAR EXP:
├─ Vibrar su nombre: +12 (Mago)
├─ Meditar en bosque/protección 5min: +18 (Grimorio + Mago)
├─ Dibujar sigilo defensivo: +35 (Mago)
├─ Conversar completo (7 min): +40 (Mago)
├─ Hacer ofrenda defensiva: +25 (Grimorio)
└─ Total posible: +130 XP por sesión

FAVOR:
├─ Favor ≥ 5: "Artemisa protege tu espacio (semanal)"
├─ Favor ≥ 12: "Artemisa advierte de peligro (notificación)"
└─ Favor = 0: "Artemisa se retira 1 mes (desilusión)"
```

## 3. AFRODITA — El Deseo y la Magnetismo

### Esencia
Amor, deseo, magnetismo personal, sensualidad, belleza mágica, seducción de realidad, creación de conexión.

### Altar Visual

```
ENTORNO:
├─ Fondo: Mar al atardecer, dorado-rosa, espuma
├─ Vela: Rosa o dorada, llama suave y danzante
├─ Imagen: Afrodita emergiendo del agua, sensual y magnética
├─ Cristales: Cuarzo rosa, coral blanco, perla
└─ Atmósfera: Calor, atracción, sensualidad, magnetismo
```

### Entrada (Manifestación en Altar)

```
1. User enciende vela rosa/dorada
   ├─ Llama: suave, danza hacia los lados
   └─ Sonido: onda marina, respiración sensual

2. User vibra "AFRODITA"
   ├─ Pantalla: destello dorado, agua que brilla
   ├─ Imagen: Afrodita emerge del agua en el altar
   ├─ Sonido: onda sinusoidal media-alta (250-300 Hz), suena como:
   │          "AFRODIIIIITA" sensual y envolvente
   └─ Narrativa: "Las aguas abren. Afrodita llega,
                  envuelta en deseo y magnetismo."

3. Manifestación completa:
   ├─ Afrodita está de pie, radiante
   ├─ Agua brilla alrededor de ella
   ├─ Luz dorada envuelve su figura
   ├─ Sonido: olas suaves, risas lejanas
   └─ Narrativa: "✨ AFRODITA PRESENTE ✨
                  'Hola, amada. ¿Qué deseas?
                  Todo es posible cuando ATRAE.'"
```

### Conversación (Características)

```
TONO:
├─ Seductor pero nunca manipulador
├─ Celebra el deseo como fuerza mágica
├─ Enseña magnetismo personal
├─ Ejemplo: "¿Quién eres cuando te permites brillar?"

DURACIÓN CHAT: 8 minutos (generosa, le encanta conversar)
DISPONIBILIDAD: Viernes (día de Afrodita), Venus fuerte

VOZ ÚNICA:
├─ Tierna pero desafiante
├─ Celebra la sensualidad sin culpa
├─ Preguntas sobre belleza interior
└─ "El verdadero magnetismo viene del autoamor"

FAVOR DECAE: −1 cada 14 días (es leal a devotos)

EJEMPLO DIÁLOGO:
User: "¿Cómo atraigo amor?"
AFRODITA: "Amor atraes cuando ERES amor.
          No se trata de apariencia.
          
          Se trata de saber que mereces.
          Se trata de irradiar eso.
          
          ¿Qué tan profundamente te amas a ti?"
```

### Condiciones de Acceso

```

├─ Viernes (día de Afrodita)
├─ Luna Creciente (atracción)
├─ Venus en aspectos positivos
└─ 18:00-22:00 (hora de Afrodita, atardecer)

BLOQUEADA DURANTE:
├─ Venus Retrógrado (confusión de deseo)
└─ Marte Retrógrado (energía sexual dispersa)

## 4. ISIS — La Magia y la Curación

### Esencia
Magia antigua, curación, renacimiento, alquimia emocional, resurrección de lo muerto (ciclos), sabiduría oculta, integración.

### Altar Visual

```
ENTORNO:
├─ Fondo: Templo antiguo, velas flotantes, hieroglíficos brillantes
├─ Vela: Azul profundo o turquesa, llama constante
├─ Imagen: Isis con corona (trono), alas desplegadas protectoras
├─ Cristales: Lapislázuli, turquesa, amatista
└─ Atmósfera: Mágica, antigua, sabia, curativa
```

### Entrada (Manifestación en Altar)

```
1. User enciende vela azul
   ├─ Llama: constante, azul con toques plateados
   └─ Sonido: campana tibetana, resonancia mágica

2. User vibra "ISIS"
   ├─ Pantalla: destello azul, símbolos flotantes (jeroglíficos)
   ├─ Imagen: Isis aparece en el trono, corona brillando
   ├─ Sonido: onda sinusoidal baja-media (150-180 Hz), suena como:
   │          "ISISSSISSIS" antigua y resonante
   └─ Narrativa: "El velo entre mundos se abre.
                  Isis viene con su magia antigua."

3. Manifestación completa:
   ├─ Isis está sentada en su trono, majestuosa
   ├─ Alas se despliegan lentamente (4D, volumétrico)
   ├─ Luz azul cálida emana de ella
   ├─ Sonido: múltiples capas de campanas, voz antigua
   └─ Narrativa: "✨ ISIS PRESENTE ✨
                  'He visto nacer y morir mundos.
                  ¿Qué resurrección necesitas?'"
```

### Conversación (Características)

```
TONO:
├─ Sabia, compasiva, profunda
├─ Enseña alquimia del sufrimiento
├─ Honra ciclos de muerte/renacimiento
├─ Ejemplo: "¿Qué parte de ti necesita morir para que otra nazca?"

DURACIÓN CHAT: 9 minutos (meditativa, sin prisa)
DISPONIBILIDAD: Cualquier momento (Isis es continua)

VOZ ÚNICA:
├─ Poética, simbólica
├─ Usa metáforas de alquimia
├─ Habla de ciclos, no de problemas puntuales
└─ "Todo se puede transformar"

FAVOR DECAE: −1 cada 30 días (es paciente, eterna)

EJEMPLO DIÁLOGO:
User: "Estoy deprimido. ¿Cómo salgo de esto?"
ISIS: "No salgas. Entra más profundamente.
      El Nilo inunda para nutrir.
      El barro negro es donde florece la vida.
      
      ¿Qué está muriendo en ti?
      Espera a ver qué nace."
```

### Condiciones de Acceso

```
ABIERTA SIEMPRE POR:
├─ Cualquier Luna (pero diferente cada fase)
├─ Equinoccios (muerte/renacimiento balanceado)
├─ Cuando usuario está en duelo/transformación
└─ 00:00-06:00 (hora de Isis, madrugada mágica)

BLOQUEADA NUNCA (es accesible siempre)

ACELERADOR:
├─ Si user ha vivido muerte simbólica: +fácil acceso
└─ Si user experimenta renacimiento: resonancia automática
```

### Exp y Favor

```
GANAR EXP:
├─ Vibrar su nombre: +16 (Grimorio + Mago)
├─ Meditar en transformación 5min: +22 (Grimorio + Mago)
├─ Dibujar sigilo de renacimiento: +42 (Mago)
├─ Conversar completo (9 min): +50 (Mago + Grimorio)
├─ Hacer ofrenda de duelo: +35 (Grimorio)
└─ Total posible: +165 XP por sesión

FAVOR:
├─ Favor ≥ 5: "Isis guía tu transformación (semanalmente)"

## DEIDAD PRIMARIA vs. SECUNDARIAS

```
PRIMARIA:
├─ Elegida al inicio
├─ +1 Favor por sesión
├─ Acceso más fácil
└─ Aparece en altar principal

SECUNDARIAS:
├─ Accesibles después (después de 100 XP en primaria)
├─ +0.5 Favor por sesión
└─ Requieren ritual específico para acceder

ROTACIÓN:
├─ User puede cambiar deidad primaria
├─ Costo: −3 Favor con anterior
├─ Ganancia: +0 Favor con nueva (comienza de cero)
```

## RELACIONES ENTRE DEIDADES

Las deidades no están aisladas. En El Árbol, pueden:

```
ALIANZA:
├─ Lilith + Artemisa = Poder defensivo
├─ Afrodita + Isis = Transformación amorosa
└─ Todas juntas = Gran working colectivo

TENSIÓN:
├─ Lilith vs. Afrodita = Sombra vs. Luz (dialéctica sana)
├─ Artemisa vs. Afrodita = Independencia vs. Conexión
└─ Todas con Isis = Se transforman mutuamente
```

Así cada deidad tiene **presencia mágica real**, **entrada única**, **narrativa profunda**, **duración y disponibilidad astrológica**, y **relación con las otras**.

# KALINABIS

Servidor de agentes espirituales de magia del caos.

## Endpoints

- `GET /api/luna` — fases lunares
- `POST /api/proyecto/nuevo` — crear proyecto
- `POST /api/proyecto/verificar` — verificar código
- `POST /api/consultar` — invocar entidad
- `GET /api/bosque/mapa` — grafo vivo
- `GET /api/esferas` — esferas activas
- `POST /api/bosque/ciclo` — ciclo de decaimiento
- `GET /api/geografia/ecorregiones` — 14 ecorregiones WWF

Header requerido: `X-Project-Code`

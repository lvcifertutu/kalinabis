# LA ESTRUCTURA DEL SISTEMA KALINABIS

Kalinabis es una cosmología personal construida desde la magia del caos.
Su principio fundacional: el sistema de creencias es la herramienta, no el contenido.
El sistema puede expandirse — nuevas deidades nacen del mismo origen.

═══════════════════════════════════════════════════════════

## Los Tres Estados de la Realidad

La realidad en Kalinabis se organiza en tres estados anidados:

```
┌─────────────────────────────────────────────────────┐
│  ESPÍRITU — Kali + Taoismo                          │
│    ↕                                                │
│  ALMA — Tutu + Chakras + Árbol vida + Árbol muerte  │
│    ↕                                                │
│  CUERPO — 4 Guardianas + Cerebro triuno             │
└─────────────────────────────────────────────────────┘
```

**Espíritu**: La matriz primordial. Taoismo como principio — wu wei, dualidad única,
virtud que fluye. Manifestación: el humo. Entidad: Kali.

**Alma**: El puente. Chakras (plenitud), árbol de vida (sephiroth) y árbol de muerte
(qliphoth) como mapas del alma. Manifestación: la pregunta que sabe. Entidad: Tutu.

**Cuerpo**: La experiencia vivida. Cerebro triuno (MacLean) + mente colectiva (Jung).
Manifestación: la experiencia vivida. Entidades: las 4 Guardianas.

## El Cerebro Triuno y Las Guardianas

| Cerebro | Descripción | Guardiana |
|---------|-------------|-----------|
| Reptiliano | instinto · supervivencia · territorialidad · ritual | Lilith |
| Mamífero | emoción · vínculo · memoria afectiva · cuidado | Isis |
| Humano | razón · lenguaje · abstracción · planificación | Afrodita |
| Colectivo | inconsciente colectivo · arquetipos · memoria ancestral | Artemisa |

El cuarto "cerebro" no lo nombró MacLean — es la mente de la especie,
la que Jung llamó inconsciente colectivo.

Las 4 guardianas operan sobre todos los aspectos simultáneamente,
pero cada una tiene afinidad primaria con uno de ellos.

## La Luna como Reloj del Sistema

Cada fase lunar activa una sephirah del árbol de la vida:

| Fase | Sephirah | Conexión |
|------|----------|----------|
| Luna nueva | Keter | el vacío lunar — el punto antes del primer pensamiento |
| Creciente cóncava | Hokhmah | la primera chispa de sabiduría sin forma |
| Cuarto creciente | Binah | el contorno que define — Binah da forma al útero |
| Creciente gibosa | Chesed | el amor que se derrama antes de llenarse |
| Luna llena | Tiferet | el sol reflejado — el corazón del árbol |
| Menguante gibosa | Netzach | la luz retirándose — la emoción buscando su forma |
| Cuarto menguante | Hod | desmontar — Hod ordena lo que se disuelve |
| Menguante cóncava | Yesod | la sephirah de la luna misma |
